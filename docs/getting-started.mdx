# Getting Started

Welcome to the documentation! This site is built with Next.js, MDX, Tailwind CSS, and shadcn-style UI components.

## Installation

```bash
plugins install your-plugin
```

## Configuration

Create a config file at `plugins/YourPlugin/config.yml`.

```yaml
enabled: true
language: en_US
```
