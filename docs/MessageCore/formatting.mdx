## Centering

Use `<center>` to center multi-line messages:

```yaml
- '[MESSAGE] <center><red>Welcome!</red><n><gray>Enjoy your stay.</gray></center>'
```

## Placeholders

Placeholders like `%player_name%` are supported via PlaceholderAPI.

```yaml
- '[MESSAGE] <green>Hello %player_name%</green>'
```
