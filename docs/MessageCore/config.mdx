---
title: Config
description: Configure MessageCore formatting, prefixes, and destinations.
---

MessageCore uses a single YAML file located at `plugins/MessageCore/config.yml`.

## Example

```yaml
prefix: "&7[&bMessageCore&7]&r "
default-destination: broadcast

formats:
  broadcast: "&a{message}"
  private: "&d[PM] &7{player}&f: &r{message}"
```

## Options

- **prefix**: Text added before all messages.
- **default-destination**: One of `broadcast` or `private`.
- **formats**: Per-destination MiniMessage/Legacy color strings.
